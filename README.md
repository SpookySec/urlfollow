# urlfollow
A small script that takes a URL and follows all redirctions
- Useful for bitly and tinyurl links

# Installation
```bash
git clone https://github.com/SpookySec/urlfollow.git
cd urlfollow
sudo bash ./install.sh
```

# Usage
```
./urlfollow --help
```
---
```
./urlfollow https://bit.ly/3o25iKq -v
```
